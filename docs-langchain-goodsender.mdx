# GoodSender Tools

GoodSender is a free email API (100,000 emails/month free) with two sending paths:
transactional templates (no consent needed) and consent-gated custom/marketing email.

This integration provides LangChain tools for sending transactional email,
custom email, and managing recipient consent via the GoodSender API.

## Overview

### Integration details

| Class | Package | Serializable | [JS support](https://js.langchain.com/docs/integrations/tools/) |
| :--- | :--- | :---: | :---: |
| GoodSenderToolkit | [langchain-goodsender](https://pypi.org/project/langchain-goodsender/) | beta | :x: |

### Tool features

| Tool | Description |
| :--- | :--- |
| `goodsender_send_template` | Send transactional email instantly (no consent needed) |
| `goodsender_send_email` | Send custom email (consent-gated) |
| `goodsender_request_consent` | Request recipient consent for custom email |

## Setup

### Credentials

Sign up at [goodsender.com](https://goodsender.com) and get your API key (free — 100K emails/month).

```bash
export GOODSENDER_API_KEY="gs_..."
```

### Installation

```bash
pip install -U langchain-goodsender
```

## Instantiation

```python
from langchain_goodsender import GoodSenderToolkit

toolkit = GoodSenderToolkit()  # reads GOODSENDER_API_KEY env var
tools = toolkit.get_tools()
```

## Invocation

### Direct

```python
from langchain_goodsender import GoodSenderToolkit

toolkit = GoodSenderToolkit()
send_template, send_email, request_consent = toolkit.get_tools()

# Send an OTP code — no consent needed
result = send_template.invoke({
    "template_id": "otp_code",
    "to_email": "alice@example.com",
    "subject": "Your verification code",
    "from_email": "noreply@myapp.com",
    "variables": {"otp_code": "482193", "app_name": "MyApp"},
})
print(result)
```

### With an agent

```python
from langchain_goodsender import GoodSenderToolkit
from langchain_openai import ChatOpenAI

toolkit = GoodSenderToolkit()
tools = toolkit.get_tools()

llm = ChatOpenAI(model="gpt-4o")
llm_with_tools = llm.bind_tools(tools)

result = llm_with_tools.invoke(
    "Send an OTP code 482193 to alice@example.com from noreply@myapp.com"
)
print(result.tool_calls)
```

## Consent caveat

Custom/marketing emails require recipient consent. Use `goodsender_request_consent`
before `goodsender_send_email`. Transactional templates (`goodsender_send_template`)
bypass consent entirely.
